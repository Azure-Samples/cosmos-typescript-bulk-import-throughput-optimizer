# Build

```
npm install
npm run build
```

# Run locally

```
export ENDPOINT_URL="https://<your-account>.documents.azure.com:443/"
export AUTHORIZATION_KEY="<your-account-key>"
node dist/main.js
```

# Run tests locally

```
npm run test
```
